# Evaluación Final Módulo 4

Aplicación web que consume la API de [JSONPlaceholder](https://jsonplaceholder.typicode.com/users) para listar y consultar información de usuarios mediante una clase en JavaScript con XMLHttpRequest.

## Funcionalidades

- Listar nombres de todos los usuarios
- Buscar información básica de un usuario (username y email)
- Buscar dirección completa de un usuario
- Buscar información avanzada (teléfono, sitio web y compañía)
- Listar todas las compañías con su catchphrase
- Listar nombres de usuarios ordenados alfabéticamente

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6+)
- XMLHttpRequest
